02 марта 2019 OLAP для 1С

Утилиты для загрузки данных в БД MS SQL Server и сводные таблицы MS Excel. Исходные тексты программы на GitHub.

Набор отчетов включает в себя

Отчеты по розничным продажам торговых точек в различных разрезах (дата, час продажи, сумма, количество, количество строк в чеке и т.п.)

Отчеты по движениям товаров и материалов (приходы, расходы, перемещения)

Остатки товаров на складах и в производстве

Выпуск продукции

Расчет себестоимости выпускаемой продукции

Иерархическая структура выпущеного продукта с себестоимостью

План-фактный анализ производства.


Процесс подготовки данных

Данные выгружаются из SQL Server базы данных 1С:ERP Управление предприятием напрямую из таблиц БД

Поддерживается выгрузка дельты данных (только изменения с последней загрузки)

Большая скорость выгрузки данных, что позволяет много раз в течение дня выгружать данные

Просмотр данных в сводных таблицах MS Excel или в кубах OLAP - в вэб приложении

Большой набор готовых отчетов (поставляется отдельно)

С небольшими модификациями может работать с любыми конфигурациями 1С.
